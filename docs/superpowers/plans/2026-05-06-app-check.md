# Firebase App Check Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Integrate Firebase App Check on Android (Play Integrity) and iOS (App Attest) so only legitimate app binaries can write to the Realtime Database.

**Architecture:** App Check is initialized at app startup on each platform before any Firebase usage. No commonMain changes are needed — gitlive SDK does not expose App Check; it's wired purely at the native layer. Debug builds use the debug provider (prints a token to logs for Firebase Console registration).

**Tech Stack:** Firebase App Check Android SDK (`firebase-appcheck-playintegrity`, `firebase-appcheck-debug`), Firebase/AppCheck CocoaPod, Play Integrity API, Apple App Attest (iOS 14+)

## Task 2: Create Android Application class with App Check initialization

**Files:**
- Create: `app/src/androidMain/kotlin/tools/mo3ta/salo/SaloApplication.kt`
- Modify: `app/src/androidMain/AndroidManifest.xml`

- [ ] **Step 1: Create SaloApplication.kt**

Create `app/src/androidMain/kotlin/tools/mo3ta/salo/SaloApplication.kt`:

```kotlin
package tools.mo3ta.salo

import android.app.Application
import com.google.firebase.appcheck.FirebaseAppCheck
import com.google.firebase.appcheck.debug.DebugAppCheckProviderFactory
import com.google.firebase.appcheck.playintegrity.PlayIntegrityAppCheckProviderFactory

class SaloApplication : Application() {
    override fun onCreate() {
        super.onCreate()
        AndroidAppContext.init(this)
        val appCheck = FirebaseAppCheck.getInstance()
        if (BuildConfig.DEBUG) {
            appCheck.installAppCheckProviderFactory(DebugAppCheckProviderFactory.getInstance())
        } else {
            appCheck.installAppCheckProviderFactory(PlayIntegrityAppCheckProviderFactory.getInstance())
        }
    }
}
```

- [ ] **Step 2: Register SaloApplication in AndroidManifest**

Open `app/src/androidMain/AndroidManifest.xml`. Add `android:name=".SaloApplication"` to the `<application>` tag:

```xml
<application
    android:name=".SaloApplication"
    android:allowBackup="true"
    android:dataExtractionRules="@xml/data_extraction_rules"
    android:fullBackupContent="@xml/backup_rules"
    android:icon="@mipmap/ic_launcher"
    android:label="@string/app_name"
    android:roundIcon="@mipmap/ic_launcher_round"
    android:supportsRtl="true"
    android:theme="@style/Theme.MyApplication">
```

- [ ] **Step 3: Update iOSApp.swift to init App Check before FirebaseApp.configure()**

Replace `iosApp/iosApp/iOSApp.swift` content:

```swift
import SwiftUI
import FirebaseCore
import FirebaseAppCheck

@main
struct iOSApp: App {
    init() {
        #if DEBUG
        let providerFactory = AppCheckDebugProviderFactory()
        AppCheck.setAppCheckProviderFactory(providerFactory)
        #else
        let providerFactory = AppAttestProviderFactory()
        AppCheck.setAppCheckProviderFactory(providerFactory)
        #endif
        FirebaseApp.configure()
    }

    var body: some Scene {
        WindowGroup {
            ContentView()
        }
    }
}
```

App Check MUST be set before `FirebaseApp.configure()`.

## Task 5: Register debug tokens in Firebase Console (manual)

This task is manual — no code changes.

- [ ] **Step 1: Open Firebase Console**

Go to Firebase Console → project `kamapp-3b3ac` → App Check → Apps.

- [ ] **Step 2: Register Android debug token**

Select the Android app → Manage debug tokens → Add token → paste the UUID from Task 2 Step 5 logcat output.

- [ ] **Step 3: Register iOS debug token**

Select the iOS app → Manage debug tokens → Add token → paste the UUID from Task 3 Step 5.

- [ ] **Step 4: Enable App Check enforcement for Realtime Database**

In Firebase Console → App Check → APIs tab → Realtime Database → click "Enforce".

> **Warning:** Once enforced, all requests without a valid App Check token are rejected. Debug builds use registered debug tokens and will continue to work. Release builds use Play Integrity / App Attest automatically.
